7️⃣ 常见问题排查

问题 原因 解决方案
Connection refused 本地 Clash 未运行或端口不匹配 检查本地代理状态与端口号
curl 卡住或超时 SSH 隧道断开 保持连接或使用 autossh 自动重连
.codex 权限错误 认证文件权限过宽被拒 执行 chmod 600 ~/.codex/auth.json
apt / git / pip 仍不走代理 未继承环境变量 在各自配置文件中单独设置代理
