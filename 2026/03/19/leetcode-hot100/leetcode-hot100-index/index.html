发表于2026-03-19|更新于2026-03-19|算法刷题LeetCode Hot100
|总字数:663|阅读时长:1分钟|浏览量:|评论数:

这套 LeetCode Hot100 系列会按“适合长期复习”的方式来整理：首页只放这一篇导航文章，单题题解不刷首页，统一在系列内部维护。

这个系列怎么读

如果你现在和我一样：

  • 刚开始系统刷题
  • Python 不是完全不会，但已经有些生疏
  • 想一边刷 Hot100，一边把语法和题型感觉捡回来

那这套系列会更偏向“从入门到能复习”的写法，而不是只放一个标准答案。

每篇题解尽量都会包含：

  • 题目到底在问什么
  • 最朴素的思路为什么成立
  • 优化思路是怎么想到的
  • Python 代码逐段解释
  • Python Tutor 可视化截图
  • 最后一段复盘总结

推荐刷题顺序

我会优先按专题推进，而不是先把所有简单题刷完。

目前比较推荐这条顺序：

  1. 哈希表
  2. 双指针
  3. 滑动窗口
  4. 普通数组 / 前缀和
  5. 栈与单调栈
  6. 二分查找
  7. 链表
  8. 二叉树遍历
  9. 回溯
  10. 堆
  11. 贪心
  12. 动态规划
  13. 图论

哈希表

  1. 两数之和：从暴力枚举到哈希表
  2. 字母异位词分组：排序 key 与计数 key
  3. 最长连续序列：排序遍历与哈希集合

双指针

  1. 移动零：额外数组到双指针原地修改
  2. 盛最多水的容器：暴力枚举到双指针夹逼
  3. 三数之和：排序去重与双指针
  4. 接雨水：前缀最大值到双指针收缩

滑动窗口

  1. 无重复字符的最长子串：从暴力到滑动窗口
  2. 找到字符串中所有字母异位词：定长滑动窗口
  3. 最小覆盖子串：可变长滑动窗口的代表题

普通数组 / 前缀和

  1. 最大子数组和：暴力到动态规划入门
  2. 合并区间：排序后逐段吞并
  3. 轮转数组：额外数组到三次反转
  4. 除自身以外数组的乘积：前缀积和后缀积
  5. 和为 K 的子数组：前缀和加哈希表

矩阵

  1. 矩阵置零：标记法与原地优化
